# Gift Abe - Notion + Supabase Integration

🎁 **Integrazione completa** tra **Notion Database** e **Supabase** per gestire i tuoi Gift Abe con supporto foto/media e **Max Flow OS**.

## 🌟 Funzionalità

✅ **Sincronizzazione Notion ↔ Supabase** bidirezionale  
✅ **Upload foto e media** diretto da Notion  
✅ **Max Flow OS** per automazioni complesse  
✅ **Real-time updates** da Supabase  
✅ **Dashboard** personalizzata  
✅ **API REST** completa  

## 📋 Requisiti

- Node.js 18+
- Notion API Key
- Supabase Account
- Max Flow OS CLI

## 🚀 Quick Start

```bash
npm install
npm run setup
npm start
```
